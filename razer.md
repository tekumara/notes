### Synapse

### System Permissions

Needed:

- Driver extensions to install the kernel drivers.
- Bluetooth for connectivity.
- Input monitoring & Accessibility - without this, Synapse can't see the mouse.
- Notifications - to show sensitivity changes.

Not needed:

- Accessibility - for macro execution.
- Screen & system audio recording - for co-ordinating the lights with whats on screen.

### Login

Razer Synapse saves your device profiles (macros, lighting setups, and key configurations) to your Razer ID account in the cloud.

### Device not connected

Must be:

- paired, or in wired mode and connected.
- input monitoring enabled.

### Lighting

Synapse can be used to configure the lighting.

Brightness - uncheck the radio button to disable lights.

Switch Off Lighting:

- When display is turned Off
- When idle for (minutes)

Smart Dimming:

Auto-dim mouse lighting during use in wireless mode to conserve battery.

0% 25% 50%

The lighting will revert to normal brightness after 3 seconds of inactivity.

### Customize buttons

Mouse button 4, 5 etc. can be customized per application.

### Mouse speed (sensitivity)

The number of dots-per-inch (DPI) of mouse movement.

Default is 1600.
