# Hex

## Viewers

Dump a file as hex:

```
xxd ~/my.hex.file
```

See also [hexyl](https://github.com/sharkdp/hexyl).

## Editors

[hexedit](http://rigaux.org/hexedit.html) can be installed via brew:

```
brew install hexedit
```

[xxd can be also used with vi](http://www.kevssite.com/2009/04/21/using-vi-as-a-hex-editor/) as a hex editor.
